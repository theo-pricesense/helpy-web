# Helpy - 추가 디자인 요청 (v2)

## 프로젝트 컨텍스트

이미 구현된 페이지:
- 랜딩 페이지 (Hero, Features, Stats, CTA, Footer)
- 로그인/회원가입 (3단계 Funnel)
- 조직 목록, 프로젝트 목록, 프로젝트 상세
- 프로필 설정

**기술 스택**: Next.js 16, React 19, Tailwind CSS v4, shadcn/ui
**테마**: 청록색(Teal/Cyan) Primary `hsl(187 80% 42%)`, 다크모드 기본

---

## 글로벌 레이아웃

### 사이드바 네비게이션 (Slack 스타일)

**목적**: 조직 간 빠른 전환 및 프로젝트 탐색

**구성요소**:
- 조직 스위처 (사이드바 상단)
  - 현재 조직 로고/이니셜 + 이름
  - 드롭다운 클릭 시 조직 목록 표시
  - 각 조직 항목: 로고, 이름, 역할 배지
  - 빠른 전환 (페이지 리로드 없이)
  - "조직 관리" 링크 (하단)
- 메인 네비게이션
  - 프로젝트 목록 (현재 조직의)
  - 각 프로젝트 클릭 시 서브메뉴 확장
    - 대화, 문서, AI 설정, 위젯, 분석
  - "새 프로젝트" 버튼
- 하단 영역
  - 사용자 프로필 (아바타 + 이름)
  - 설정 링크
  - 로그아웃

**동작**:
- 조직 전환 시 URL 구조: `/dashboard/[orgId]/...`
- 선택된 프로젝트 하이라이트
- 사이드바 축소/확장 토글 (모바일 대응)

---

## 디자인 요청 페이지

### 1. 조직 설정 페이지 (`/settings/organization/[orgId]`)

**목적**: 조직 정보 수정 및 멤버 관리

**구성요소**:
- 조직 정보 카드
  - 조직명 수정 폼
  - 조직 생성일 표시
- 멤버 관리 섹션
  - 멤버 목록 테이블 (이름, 이메일, 역할, 가입일)
  - 역할: Owner, Admin, Member
  - "멤버 초대" 버튼 → 이메일 입력 모달
  - 멤버 제거 버튼 (확인 모달)
- Danger Zone
  - 조직 삭제 버튼 (확인 모달)

---

### 2. 프로젝트 AI 설정 페이지 (`/projects/[projectId]/ai`)

**목적**: AI 챗봇 동작 설정

**구성요소**:
- AI 모델 설정 카드
  - 모델 선택 드롭다운 (GPT-4o, Claude 3.5 등)
  - Temperature 슬라이더 (0.0 ~ 1.0)
  - Max tokens 입력
- 시스템 프롬프트 카드
  - 대형 텍스트에어리어
  - 플레이스홀더: "AI가 어떻게 응답할지 지시하세요"
  - 저장 버튼
- 응답 스타일 카드
  - 톤 선택: 친근함 / 전문적 / 중립
  - 언어: 한국어, 영어, 자동감지
- 테스트 채팅 미니 패널
  - 간단한 채팅 UI로 설정 테스트

---

### 3. 문서 관리 페이지 (`/projects/[projectId]/documents`)

**목적**: AI 지식 베이스 문서 관리

**구성요소**:
- 문서 업로드 영역
  - 드래그 앤 드롭 존
  - 지원: PDF, TXT, MD, DOCX
  - 업로드 진행률
- 문서 목록 테이블
  - 파일명, 크기, 업로드일, 상태(처리중/완료/실패)
  - 삭제 버튼
- 웹 크롤링 섹션
  - URL 입력 → 크롤링 버튼
  - 크롤링된 페이지 목록

---

### 4. 대화 목록 페이지 (`/projects/[projectId]/conversations`)

**목적**: 고객 대화 내역 조회

**구성요소**:
- 검색/필터 바
  - 검색 입력
  - 상태 필터: 전체/진행중/종료
  - 날짜 범위
- 대화 목록 테이블
  - 고객 ID, 마지막 메시지 미리보기, 시작시간, 메시지 수, 상태
- 페이지네이션

---

### 5. 대화 상세 페이지 (`/projects/[projectId]/conversations/[id]`)

**목적**: 개별 대화 확인 및 상담사 개입

**구성요소**:
- 헤더: 고객 정보, 상태 배지, "상담사 모드" 버튼
- 채팅 뷰
  - 메시지 버블 (고객 왼쪽, AI/상담사 오른쪽)
  - AI 응답에 "AI" 배지
  - 타임스탬프
- 메시지 입력 (상담사 모드)

---

### 6. 분석 대시보드 (`/projects/[projectId]/analytics`)

**목적**: 프로젝트 성과 지표

**구성요소**:
- 기간 선택: 오늘/7일/30일/커스텀
- 핵심 지표 카드 4개
  - 총 대화 수, AI 해결률, 평균 응답 시간, 고객 만족도
- 차트
  - 일별 대화 추이 (라인)
  - 시간대별 대화량 (바)
  - AI vs 상담사 비율 (도넛)
- 자주 묻는 질문 Top 10

---

### 7. 위젯 설정 페이지 (`/projects/[projectId]/widget`)

**목적**: 채팅 위젯 커스터마이징

**구성요소**:
- 실시간 위젯 미리보기 (모바일/데스크탑 토글)
- 스타일 설정
  - 테마 컬러 (컬러 피커)
  - 위치: 좌하단/우하단
  - 환영 메시지
- 행동 설정
  - 자동 열기 딜레이
  - 업무 시간
- 설치 코드 스니펫 + 복사 버튼

---

### 8. 멤버 초대 모달

**구성요소**:
- 이메일 입력
- 역할 선택 (Admin/Member)
- 초대 버튼
- 대기 중인 초대 목록

---

### 9. 글로벌 검색 (Command Palette)

**단축키**: `⌘ + K` (Mac) / `Ctrl + K` (Windows)

**구성요소**:
- 중앙 모달 (반투명 배경)
- 검색 입력 필드 (자동 포커스)
- 검색 결과 섹션
  - 대화 (고객 ID, 내용 미리보기)
  - 문서 (제목)
  - 프로젝트
  - 설정 페이지
  - 빠른 액션 (새 프로젝트, 문서 업로드 등)
- 최근 검색 기록
- 키보드 네비게이션 (↑↓ 이동, Enter 선택, Esc 닫기)

---

### 10. 알림 센터

**UI 위치**: 헤더 우측 벨 아이콘

**구성요소**:
- 알림 드롭다운 패널
- 알림 유형별 아이콘/색상
  - 에스컬레이션 (주황)
  - 낮은 만족도 (빨강)
  - 문서 처리 완료 (녹색)
  - 팀 멘션 (파랑)
- 읽음/안읽음 상태
- "모두 읽음 처리" 버튼
- 알림 설정 링크

---

### 11. 온보딩 위자드

**목적**: 신규 사용자 첫 설정 가이드

**단계** (스텝퍼 UI):
1. 조직 생성 - 이름 입력
2. 첫 프로젝트 생성 - 이름, 설명
3. AI 기본 설정 - 톤, 언어 선택
4. 문서 업로드 (선택) - 드래그 앤 드롭
5. 위젯 미리보기 - 테스트 채팅, 설치 코드

**구성요소**:
- 진행률 표시 (1/5, 2/5...)
- "나중에 하기" 스킵 버튼
- 각 단계 완료 체크 표시

---

### 12. 가격 페이지 (`/pricing`)

**목적**: 플랜 비교 및 결제 유도

**구성요소**:
- 플랜 카드 3개 (Starter / Pro / Enterprise)
  - 가격 표시 (월/연간 토글)
  - 기능 목록 체크리스트
  - CTA 버튼 ("시작하기", "영업팀 문의")
- Pro 플랜 "추천" 배지 강조
- FAQ 섹션 (아코디언)
- "베타 파트너 모집 중" 배너 (선택)

---

## 공통 스타일 가이드

- **Primary**: `hsl(187 80% 42%)` (Teal/Cyan)
- **배경**: 다크모드 `#0d1017`, 라이트모드 `#f5f7fa`
- **카드**: `bg-card`, `border-border`, 둥근 모서리
- **버튼**: Primary(청록), Outline, Destructive(빨강), Ghost
- **아이콘**: Lucide React
- **폰트**: Inter (본문), JetBrains Mono (코드)
