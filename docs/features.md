# Helpy - Feature Specification

AI-first 고객 서비스 플랫폼의 핵심 기능 정의

---

## 1. AI 핵심 기능

### 1.1 AI 피드백 & 학습 시스템

**목적**: 상담사 피드백을 통한 AI 응답 품질 지속 개선

**기능**:
- 각 AI 응답에 👍/👎 피드백 버튼
- 잘못된 응답 수정 기능 (올바른 답변 입력)
- 피드백 기반 자동 재학습 파이프라인
- 피드백 통계 대시보드 (개선율 추이)

**UI 위치**: 대화 상세 페이지, 각 AI 메시지 버블

---

### 1.2 지식 갭 감지 (Knowledge Gap Detection)

**목적**: AI가 답변하지 못하는 질문 영역 파악

**기능**:
- 미응답/낮은 신뢰도 질문 자동 수집
- 주제별 클러스터링
- "문서 추가 필요" 알림
- 추천 문서 주제 제안

**UI 위치**: 문서 관리 페이지, 분석 대시보드

---

### 1.3 AI 신뢰도 표시 (Confidence Score)

**목적**: AI 응답의 확실성을 투명하게 표시

**기능**:
- 각 응답에 신뢰도 % 표시 (예: 92%)
- 신뢰도 기준:
  - 90%+ : 높음 (녹색)
  - 70-89%: 중간 (노란색)
  - 70% 미만: 낮음 (빨간색)
- 낮은 신뢰도 시 면책 메시지 자동 추가

**UI 위치**: 대화 상세 (상담사 뷰), 위젯 (선택적)

---

### 1.4 자동 에스컬레이션 규칙

**목적**: AI가 처리하기 어려운 상황에서 상담사에게 자동 전달

**에스컬레이션 트리거**:
- 신뢰도 임계값 미만 (설정 가능, 기본 50%)
- 고객 감정 부정적 감지
- 특정 키워드 감지 (환불, 클레임, 법적 등)
- 고객이 "상담사 연결" 요청
- 동일 질문 N회 반복

**설정 옵션**:
- 트리거별 활성화/비활성화
- 임계값 조정
- 알림 채널 (이메일, 슬랙, 인앱)

**UI 위치**: 프로젝트 AI 설정 페이지

---

### 1.5 대화 요약 (Conversation Summary)

**목적**: 긴 대화를 빠르게 파악

**기능**:
- AI 자동 요약 생성 (실시간)
- 요약 포함 내용:
  - 고객 문의 주제
  - 해결 여부
  - 주요 논의 사항
  - 필요 후속 조치
- 대화 종료 시 최종 요약 저장

**UI 위치**: 대화 목록 (미리보기), 대화 상세 (사이드패널)

---

### 1.6 인텐트 분석 (Intent Analytics)

**목적**: 고객 질문 패턴 파악 및 서비스 개선

**기능**:
- 자동 인텐트 분류
  - 제품 문의, 주문/배송, 환불/교환, 기술 지원, 계정 문제, 기타
- 인텐트별 통계
  - 빈도, AI 해결률, 평균 대화 시간
- 트렌드 변화 알림
- 커스텀 인텐트 정의

**UI 위치**: 분석 대시보드

---

### 1.7 상담사 AI 어시스턴트

**목적**: 상담사가 직접 응대할 때 AI가 보조

**기능**:
- 실시간 응답 제안 (3개 옵션)
- 관련 문서 자동 검색 & 표시
- 고객 정보 요약 (이전 대화 내역)
- 매크로/템플릿 제안

**UI 위치**: 대화 상세 페이지 (상담사 모드)

---

### 1.8 감정 분석 (Sentiment Analysis)

**목적**: 고객 감정 상태 파악 및 대응

**기능**:
- 실시간 감정 감지: 긍정/중립/부정
- 대화별 감정 추이 그래프
- 부정 감정 급증 시 알림
- 감정 기반 우선순위 정렬

**UI 위치**: 대화 목록 (아이콘), 대화 상세 (사이드패널)

---

## 2. 사용자 경험 (UX) 기능

### 2.1 온보딩 위자드

**목적**: 신규 사용자의 빠른 시작

**단계**:
1. 조직 생성
2. 첫 프로젝트 생성
3. AI 기본 설정 (톤, 언어)
4. 문서 업로드 (선택)
5. 위젯 미리보기 & 설치 코드

**특징**:
- 단계별 진행률 표시
- "나중에 하기" 옵션
- 체크리스트 형태로 남은 작업 표시

---

### 2.2 글로벌 검색 (Command Palette)

**목적**: 빠른 탐색 및 액션

**단축키**: `⌘ + K` (Mac) / `Ctrl + K` (Windows)

**검색 대상**:
- 대화 (고객 ID, 내용)
- 문서 (제목, 내용)
- 프로젝트
- 설정 페이지
- 빠른 액션 (새 프로젝트, 문서 업로드 등)

**UI**: 모달 형태, 최근 검색 기록

---

### 2.3 알림 센터

**목적**: 중요 이벤트 실시간 알림

**알림 유형**:
- 에스컬레이션 발생
- 고객 만족도 낮음 (별점 1-2)
- 새 대화 시작 (선택적)
- 문서 처리 완료/실패
- 팀원 멘션

**채널**:
- 인앱 알림 (벨 아이콘)
- 이메일 (설정 가능)
- 슬랙/디스코드 웹훅

**UI 위치**: 헤더 우측, 알림 드롭다운

---

### 2.4 프롬프트 템플릿 갤러리

**목적**: 빠른 AI 설정 시작

**제공 템플릿**:
- 이커머스 고객 지원
- SaaS 기술 지원
- 예약/스케줄링 서비스
- 교육/코스 안내
- 일반 FAQ

**기능**:
- 템플릿 미리보기
- 원클릭 적용
- 커스텀 템플릿 저장
- 팀 템플릿 공유

**UI 위치**: 프로젝트 AI 설정 페이지

---

### 2.5 키보드 단축키

**목적**: 파워 유저 생산성 향상

**주요 단축키**:
| 단축키 | 동작 |
|--------|------|
| `⌘ + K` | 글로벌 검색 |
| `⌘ + N` | 새 프로젝트 |
| `⌘ + /` | 단축키 도움말 |
| `←` / `→` | 대화 목록 이전/다음 |
| `E` | 상담사 모드 토글 |
| `R` | 새로고침 |

**UI**: 단축키 도움말 모달

---

### 2.6 다크/라이트 모드 토글

**목적**: 사용자 선호 테마 지원

**옵션**:
- 라이트 모드
- 다크 모드 (기본)
- 시스템 설정 따르기

**UI 위치**: 사이드바 하단 또는 설정

---

## 3. 통합 & 확장

### 3.1 멀티채널 지원

**지원 채널**:
- 웹 위젯 (기본)
- 슬랙
- 디스코드
- 카카오톡 (향후)
- 이메일 (향후)

**기능**:
- 채널별 프로젝트 연결
- 통합 대화 인박스
- 채널 표시 배지

---

### 3.2 웹훅 & API

**목적**: 외부 시스템 연동

**웹훅 이벤트**:
- 새 대화 시작
- 대화 종료
- 에스컬레이션 발생
- 메시지 수신

**API**:
- REST API (인증: API Key)
- 대화 생성/조회
- 메시지 전송
- 문서 업로드

---

### 3.3 CRM 연동

**지원 CRM**:
- Salesforce
- HubSpot
- Zendesk
- 커스텀 웹훅

**동기화 데이터**:
- 고객 정보
- 대화 기록
- 티켓 생성

---

## 4. 보안 & 관리

### 4.1 역할 기반 접근 제어 (RBAC)

**역할**:
| 역할 | 권한 |
|------|------|
| Owner | 모든 권한 + 조직 삭제 |
| Admin | 프로젝트 관리, 멤버 초대 |
| Member | 대화 조회/응대, 문서 조회 |

---

### 4.2 감사 로그 (Audit Log)

**기록 대상**:
- 로그인/로그아웃
- 멤버 초대/제거
- 프로젝트 생성/삭제
- AI 설정 변경
- API 키 재생성

**보관**: 90일

---

### 4.3 데이터 내보내기

**내보내기 가능 데이터**:
- 대화 기록 (CSV, JSON)
- 분석 리포트 (PDF)
- 문서 목록

---
